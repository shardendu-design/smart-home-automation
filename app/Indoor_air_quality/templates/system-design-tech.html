Frontend



User Interface (UI):

  • Technologies: HTML, CSS, Bootstrap, and JavaScript.
  • Design: A clean and intuitive dashboard that displays real-time air quality data, historical trends, and alerts. It includes interactive charts created with JavaScript.

Data Visualization:

  • Approach: Display data in various formats such as graphs, charts, and maps for easy interpretation.
  • Tools: Advanced data visualization is achieved using tools like Matplotlib.

Notification System:

  • Functionality: Alerts users about device automation via email.

Accessibility and Responsiveness:

  • Compatibility: The application is designed to be accessible on various devices, including desktops, tablets, and smartphones, and conforms to web accessibility standards.


Backend



Programming Language and Framework:

  • Technology Used: Core Python and Flask are used for data analysis and backend development.

Database:

  • Structure: Three different data types are used to store sensor data and machine learning prediction data in CSV, JSON, and PostgreSQL containers through Raspberry Pi 4. CSV and JSON data are saved on an external hard drive connected to the Raspberry Pi.

Data Collection and Processing:

  • Procedure: The backend collects data from sensors, preprocesses it, and deploys a machine learning model for real-time predictions.

Hosting:

  • Current Stage: Port forwarding is currently in use for testing application behavior across different networks. In the future, the plan is to use a personal domain with DNS and SSL certificates for enhanced security.

Security:

  • Protection: The entire network is secured with DPN (Deeper Connect Decentralized VPN), ensuring protection for all IoT devices and data.

Scalability:

  • Storage Capacity: A 500 GB external hard drive is currently used for data storage, with the option to scale up as needed.

Overall Architecture:

  • Data Flow: Sensors → Data Processing (Backend) → Database → Data Retrieval (API) → Data Visualization (Frontend).
  • Scalability and Reliability: Utilizing port forwarding, a public IP through a domain name with an SSL certificate, DPN, and expandable data storage capacity.
  • Security: Ensuring secure data transfer and storage, coupled with user authentication and authorization mechanisms through DPN.
